Detailed Description

The goal of the class is to look for the words in the provided .txt file. Finder reads FSA from .json into the variable and performs the search, outputting all the occurences of the words in the file along with given lines. Public constructor is called when defining a class object.

Parameters
jsonThe file to read FSA from.
searchThe file to perform a search on.
wVocabulary of the words to look for in a search file.

Function reads FSA saved in json file into the class variables. Assumes json looks like the following: 

      index: 4,
      final: 1,
      fail:  3,
      transitions:{
      t : 5
      },
      outputs:[
      nicht
      ]

Function results in an error if the file is corrupted or doesn't exist.
